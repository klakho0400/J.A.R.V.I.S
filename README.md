# J.A.R.V.I.S

## Basic implementation of a voice assistants

* Weather
* Date 
* Time
* Corona cases in India
* Flip a coin
* Roll a dice
* Open Websites
* Play Songs
* Write emails
* Play Games
* Store Notes
* Wiki Search
* Find Location
* And many more to be added ;p
